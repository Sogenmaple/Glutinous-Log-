# 🖱️ 自定义光标系统 - 极简科技风格

## ✨ 功能特性

### 1. 多状态光标
网站使用了定制的 SVG 光标，根据不同交互场景自动切换：

| 场景 | 光标样式 | 颜色 |
|------|---------|------|
| **默认** | 圆点 + 外环 | 琥珀色 |
| **链接** | 旋转外环圆点 | 青色 |
| **按钮** | 双环圆点 | 琥珀色 |
| **输入框** | 竖线光标 | 青色 |
| **导航** | 十字圆点 | 白色 |
| **卡片悬停** | 三环扩散 | 琥珀色 |
| **代码块** | `<>` 符号 | 青色 |
| **加载中** | 旋转四点 | 琥珀色 |
| **禁用** | 灰色圆点 | 灰色 |
| **后台管理** | 方形圆点 | 青色 |

### 2. 粒子拖尾效果
- 鼠标移动时产生琥珀色和青色的粒子拖尾
- 粒子受重力影响自然下落
- 点击时产生粒子爆发效果（12 个粒子）
- 悬停在可交互元素上时产生额外粒子

### 3. 点击波纹
- 每次点击都会产生琥珀色波纹扩散效果
- 波纹带有发光效果
- 自动淡出消失

### 4. 悬停增强
- 悬停在卡片、链接、按钮上时光标会略微放大
- 游戏卡片区域有特殊的光环效果

## 🎨 设计细节

### 颜色方案
```css
--cursor-color: #ffffff;        /* 白色 */
--cursor-accent: #ff9500;       /* 琥珀色 - 强调色 */
--cursor-size: 24px;            /* 光标尺寸 */
```

### 设计理念
- **极简主义** - 圆形几何元素，去除多余装饰
- **细腻优雅** - 小尺寸（24px），精致不抢眼
- **科技感** - 旋转、扩散等动态效果
- **柔和配色** - 白色/琥珀色/青色，和谐统一

## 🚀 技术实现

### 1. CSS 光标（内联 SVG）
```css
* {
  cursor: url("data:image/svg+xml;utf8,<svg>...</svg>") 0 0, auto;
}
```

### 2. 粒子系统（React + Vanilla JS）
- 使用 `requestAnimationFrame` 实现流畅动画
- 粒子有生命周期、速度、重力等物理属性
- 自动清理过期粒子避免内存泄漏

### 3. 响应式支持
```css
@media (hover: none) and (pointer: coarse) {
  * {
    cursor: auto !important;
  }
  
  #cursor-particles {
    display: none !important;
  }
}
```
移动端自动禁用自定义光标和粒子效果，使用系统默认光标。

## 🎯 交互场景

### 粒子生成规则
- **移动时**: 根据移动速度生成 1-3 个粒子/帧
- **点击时**: 环形分布 8 个粒子 + 波纹效果
- **悬停可交互元素**: 环形分布 6 个粒子

### 性能优化
- 仅在桌面端启用（检测 `hover` 和 `pointer`）
- 粒子数量自动限制
- 使用 CSS `transform` 和 `opacity` 实现 GPU 加速
- 动画帧率自适应

## 🔧 自定义配置

### 调整粒子密度
```jsx
// 在 CursorEffect.jsx 中修改概率值
if (Math.random() < 0.6) {  // 改为 0.3 减少密度，0.9 增加密度
  createParticle(mouseX, mouseY)
}
```

### 修改光标颜色
```css
/* 在 cursor.css 中修改 CSS 变量 */
:root {
  --cursor-primary: #你的颜色;
  --cursor-secondary: #你的颜色;
}
```

### 禁用粒子效果
```jsx
// 在 App.jsx 中注释掉 CursorEffect 组件
// <CursorEffect />
```
