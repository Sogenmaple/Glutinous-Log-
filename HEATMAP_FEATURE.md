# 🔥 热力图功能说明

## 📊 新增功能 (v1.2.0)

### 1. 专注热力图 🔥

#### 功能描述
类似 GitHub 贡献图的可视化热力图，展示最近 42 天（6 周）的专注分布情况。

#### 视觉效果

```
│ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ │  □ 0
│ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ │  □ 1-2
│ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ │  □ 3-4
│ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ □ │  □ 5-6

```

#### 颜色分级
| 专注数量 | 颜色 | 透明度 |
|---------|------|--------|
| **0** | 白色 | 5% |
| **1-2** | 琥珀色 | 30% |
| **3-4** | 琥珀色 | 50% |
| **5-6** | 琥珀色 | 70% |
| **7+** | 琥珀色 | 90% |

#### 交互效果
- **悬停放大**: 方块放大 1.3 倍
- **边框高亮**: 琥珀色边框发光
- **阴影效果**: 橙色光晕
- **Tooltip**: 显示日期和专注数量

#### 数据展示
- **横向**: 42 天（6 周）
- **纵向**: 7 行（代表一周 7 天）
- **总计**: 294 个方块
- **月份标注**: 底部显示月份

### 2. 实时统计 📈

#### 展示内容
```
┌─────────────────────────────────────┐
│  今日专注    本周专注    总计专注   │
│     8          24         156      │
│  个番茄钟   个番茄钟   个番茄钟   │
└─────────────────────────────────────┘
```

#### 特点
- **横向布局**: 三个统计项并排显示
- **实时更新**: 完成番茄钟后立即更新
- **紧凑设计**: 节省空间，信息清晰
- **视觉层次**: 数值 > 标签 > 单位

#### 样式
- 背景：半透明白色
- 边框：琥珀色光晕
- 字体：数字使用 display 字体
- 颜色：琥珀色数值 + 灰色标签

### 3. 年月日倒计时 ⏳

#### 倒计时项目
| 项目 | 说明 | 格式 |
|------|------|------|
| **距离新年** | 距离下一年 1 月 1 日 | X 天 XX 时 XX 分 XX 秒 |
| **距离下月** | 距离下个月 1 号 | X 天 XX 时 XX 分 XX 秒 |
| **距离明天** | 距离明天 0 点 | X 天 XX 时 XX 分 XX 秒 |

#### 视觉效果
```
┌──────────────────────────────────────┐
│ 距离新年   125 天 08 时 32 分 15 秒   │
│ 距离下月    12 天 14 时 20 分 08 秒   │
│ 距离明天     0 天 09 时 27 分 42 秒   │
└──────────────────────────────────────┘
```

#### 技术实现
- **每秒更新**: 使用 setInterval 实时刷新
- **精确计算**: 基于时间戳差值
- **自动进位**: 天/时/分/秒自动转换

## 💡 使用技巧

### 热力图解读
1. **颜色越深** = 专注越多
2. **空白方块** = 没有专注记录
3. **连续深色** = 保持良好习惯
4. **断档明显** = 需要加强自律

### 倒计时用途
- **年倒计时**: 年度目标进度参考
- **月倒计时**: 月度计划时间管理
- **日倒计时**: 珍惜每一天

### 数据持久化
- 热力图数据自动保存到 localStorage
- 刷新页面不丢失历史记录
- 导出功能包含热力图数据
