# 🍅 番茄钟待办 - 功能更新

## 📦 新增功能 (v1.1.0)

### 1. 本地存储 💾
所有数据自动保存到浏览器本地存储，刷新页面不丢失！

**保存内容**:
- ✅ 待办事项列表
- ✅ 统计数据
- ✅ 当前番茄钟进度
- ✅ 当前模式（专注/休息）
- ✅ 完成的番茄钟数量

**存储键名**:
```javascript
localStorage.setItem('pomodoro_todos', ...)
localStorage.setItem('pomodoro_stats', ...)
localStorage.setItem('pomodoro_timeLeft', ...)
localStorage.setItem('pomodoro_mode', ...)
localStorage.setItem('pomodoro_completed', ...)
```

### 2. 数据统计 📊

#### 统计维度
| 统计项 | 说明 |
|--------|------|
| **今日** | 今天完成的番茄钟数量 |
| **本周** | 本周完成的番茄钟数量 |
| **总计** | 历史累计完成的番茄钟总数 |
| **最佳记录** | 单日完成最多的记录和日期 |

#### 统计更新逻辑
- 每次完成专注模式自动 +1
- 自动检测并更新最佳记录
- 实时显示在番茄钟下方

#### 网格布局
```
┌─────────┬─────────┐
│ 今日    │ 本周    │
│   8     │   24    │
│ 个番茄钟│ 个番茄钟│
├─────────┼─────────┤
│ 总计    │ 最佳记录│
│   156   │   12    │
│ 个番茄钟│ 03-05   │
└─────────┴─────────┘
```

### 3. 任务优先级 🎯

#### 三个等级
| 优先级 | 颜色 | 标识 |
|--------|------|------|
| **高** | 红色 (#ff453a) | 左边框红色 + 红色徽章 |
| **中** | 琥珀色 (#ff9500) | 左边框琥珀色 + 琥珀色徽章 |
| **低** | 青色 (#06b6d4) | 左边框青色 + 青色徽章 |

#### 使用方式
1. 在添加任务时选择优先级
2. 任务卡片左侧显示颜色条
3. 任务文字旁显示优先级徽章
4. 高优先级任务更醒目

---

### 4. 数据导出 📤

#### 导出功能
- 点击「导出」按钮
- 自动生成 JSON 文件
- 文件名：`pomodoro-backup-YYYY-MM-DD.json`
- 包含内容：待办列表 + 统计数据 + 完成数

#### 导出数据结构
```json
{
  "todos": [
    {
      "id": 1234567890,
      "text": "完成项目报告",
      "completed": false,
      "priority": "high",
      "createdAt": "2026-03-06T10:30:00.000Z"
    }
  ],
  "stats": {
    "today": 8,
    "week": 24,
    "total": 156,
    "bestDay": {
      "date": "2026-03-05",
      "count": 12
    }
  },
  "completedPomodoros": 8
}
```

### 5. 清空数据 🗑️

#### 功能说明
- 点击「清空」按钮
- 二次确认防止误操作
- 清空所有数据并重置
- 同时清除本地存储

#### 重置内容
- ✅ 待办列表 → 空
- ✅ 统计数据 → 归零
- ✅ 完成数 → 0
- ✅ 专注时间 → 0
- ✅ 计时器 → 25 分钟
- ✅ 模式 → 专注模式

## 🚀 使用流程

### 日常使用
1. **打开页面** → 自动加载上次进度
2. **添加任务** → 选择优先级 → 点击添加
3. **设置专注** → 点击任务的 ▶️ 按钮
4. **开始番茄钟** → 点击「开始」
5. **完成专注** → 自动统计 +1
6. **导出数据** → 定期备份数据

### 数据管理
- **自动保存**: 每次操作自动保存
- **手动导出**: 点击导出按钮备份
- **清空重置**: 确认清空所有数据

---

## 💡 使用技巧

### 1. 优先级管理
- 🔴 **高优先级**: 重要且紧急的任务
- 🟠 **中优先级**: 重要但不紧急
- 🔵 **低优先级**: 可以稍后处理

### 2. 专注策略
- 早上开始第一个番茄钟
- 每 4 个番茄钟后休息 15 分钟
- 高优先级任务安排在精力最好的时段

### 3. 数据备份
- 每周导出一次数据
- 更换设备前记得导出
- 导出的 JSON 可以手动编辑
