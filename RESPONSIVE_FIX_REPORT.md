|---------|---------|---------|
| 小屏手机 | <480px | 单列布局、紧凑间距、隐藏装饰 |

| 笔记本 | 1025-1365px | 标准布局 |
| 桌面 | 1366-1919px | 优化间距和字体 |
| 大屏桌面 | 1920-2559px | 最大宽度限制、更大间距 |
| 超大屏 | 2560px+ | 2K/4K优化、字体放大 |
| 带鱼屏 | 21:9+ | 内容宽度限制 |
| 竖屏笔记本 | 4:5- | 垂直布局优化 |

## 关键技术点

1. **clamp() 函数**: 实现平滑的字体和尺寸缩放
   ```css
   font-size: clamp(1rem, 2vw, 1.5rem);
   ```

2. **auto-fit + minmax()**: 自适应网格布局
   ```css
   grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
   ```

3. **相对单位**: 使用 vw、vh、rem 代替固定 px

4. **媒体查询**: 针对不同设备类型优化

5. **触摸优化**: 增大触摸目标（最小 44px）

6. **性能优化**: 移动端简化动画和装饰
